MyAgent-IM - программа для обмена сообщениями по протоколу mail.ru

Для компиляции требуются:
1) g++
2) flex
3) pkg-config

библиотеки:
1) libqt4-dev (>= 4.4)
2) zlib1g-dev
3) libxapian-dev
4) libaspell-dev
5) libxss-dev
6) libphonon-dev

Компиляция:
> qmake
> make

Инсталляция (под рутом):
> make install

Бинарник копируется в /usr/local/bin, файлы данных (переводы) в /usr/local/share/myagent-im
Изменить пути можно подредактировав файл проекта src.pro (переменные BINDIR и DATADIR)

При компиляции из сборок ВСЕГДА делайте `make uninstall' предыдущей версии.

Запуск:
> myagent-im

Это альфа-версия, так что ничего не обещаю, ни за что не отвечаю ;)

-------------------------------------------------------------------
Часто встречающиеся проблемы:

Если qmake выводит предупреждения, то проверьте версию Qt.
Она должна быть не меньше чем 4.4.
Узнать версию можно выполнив qmake -v
Должно вывестись примерно следующее:
QMake version 2.01a
Using Qt version 4.4.0 in /usr/lib
